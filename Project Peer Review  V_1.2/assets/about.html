Frequently Asked Questions About Project Peer Review


Q. What is Project Peer Review?

A. Project Peer Review is a project notes tracker for developers to help keep them up to date on progress.


Q. How do I run the app?

A. Simply open the index.html file in your browser of choice, no server needed


Q. What API is it using?

A. This app uses the Firebase API.


Q. Is this app cross platform compatible?

A. Yes this is a scalable app that will run on multiple devices


Q. How are the comments displayed?

A. The comments list is displayed in chronological order from oldest (top) to newest(bottom)


Q. If you delete a comment by a user, can the user still post comments

A. Yes when a comment by a user is deleted, the user is not deleted from the user list. The user stays in the dropdown list for comment owner until they are deleted from the user list.
